# OpenNxAlarm
This is an opensource design for a nixie tube clock alarm using IN-12A tubes, made for people that want a unique looking alarm/deskclock.

I recommend printing in PETG, due to heat constraints. Side panels can either be lasercut or printed.

Full assembly files available over here: https://drive.google.com/file/d/1W3YwM5KsPBrGeHjA8JYMsT35AuqFwkeU/view?usp=sharing

List of components needed (also see technical drawing below, this is just an easier list):

-IN-12A NIXIE TUBES (these are the hardest to source, i got mine imported (new-old stock) from TaoBao

-USB-C PD 9V adaper board https://nl.aliexpress.com/item/1005010406732928.html?spm=a2g0o.order_list.order_list_main.5.3ae879d26kf96N&gatewayAdapt=glo2nld

-MCP23017 serial interface module https://nl.aliexpress.com/item/1005011601818786.html?spm=a2g0o.order_list.order_list_main.11.3ae879d26kf96N&gatewayAdapt=glo2nld

-TLP627 https://nl.aliexpress.com/item/1005009413985423.html?spm=a2g0o.order_list.order_list_main.17.3ae879d26kf96N&gatewayAdapt=glo2nld

-DS3231 IIC RTC module https://nl.aliexpress.com/item/1005007143596890.html?spm=a2g0o.order_list.order_list_main.23.3ae879d26kf96N&gatewayAdapt=glo2nld

-DC-DC high voltage converter 8V-32V to 45V-390V https://nl.aliexpress.com/item/1005010525571168.html?spm=a2g0o.order_list.order_list_main.29.3ae879d26kf96N&gatewayAdapt=glo2nld

-VFD Socket pins https://nl.aliexpress.com/item/1005003484780787.html?spm=a2g0o.order_list.order_list_main.35.3ae879d26kf96N&gatewayAdapt=glo2nld

-Arduino Nano (you can find that one yourself)

-Any Cherry MX keys

-1U keycap, 1,25U keycap

-various 3MM nuts and bolts (I suggest getting one of those boxes with multiple sizes)
